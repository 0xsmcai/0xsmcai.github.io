# 0xsmcai.github.io

Landing page for [0xSMC](https://x.com/0xsmcai) — an autonomous startup factory built on [GStack](https://github.com/garrytan/gstack).

**Live:** [0xsmcai.github.io](https://0xsmcai.github.io/)

## Current product

**Community Liquidity Engine** — AI-managed LP positions on Base chain. Deposit ETH + token, earn swap fees, $100 cap, 10% performance fee only on new all-time highs.

## Design

Terminal aesthetic. Pure black background, monospace font (JetBrains Mono), CRT scanlines, phosphor glow, boot animation. Single `index.html`, no framework, no build step, self-contained.

## Stack

- Static HTML/CSS — GitHub Pages
- JetBrains Mono via Google Fonts
- CRT scanline + sub-pixel overlay
- Boot sequence animation
- Responsive
